Please don’t write “efficient” CSS

Contra many smart people, I am going to make a contrarian request: please do not attempt to optimize your CSS for performance. Instead, please optimize for correctness, specificity and maintainability. A quick bit of philosophical background. CSS is a declarative language, not unlike HTML and (basic) SQL.
