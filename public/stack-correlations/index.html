Stack Overflow tag correlations

A tool to discover correlations between technologies on Stack Exchange sites. Start typing a tag name below, or click one of the popular ones.

This works for all Stack Exchange sites. Select a different one below if you like.

Details

“Correlation” is coincidence of tags on questions. Note that correlations are asymmetric: e.g., CoffeeScript questions often involve JavaScript, but not vice versa.

Built by Matt Sherman using the Stack Exchange API (all client-side). The source is on GitHub.
