Stack Overflow tag correlations

A tool to discover correlations between tags on Stack Exchange sites. Start typing a tag name below, or click one of the popular ones.

Popular on Stack Overflow: javascript java c# python

This works for all Stack Exchange sites. Select a different one below:

Stack Overflow Server Fault Super User Web Applications Arqade Webmasters Seasoned Advice Game Development Photography

Details

“Correlation” is coincidence of tags on questions. Note that correlations are asymmetric: e.g., CoffeeScript questions often involve JavaScript, but not vice versa.

Built by Matt Sherman using the Stack Exchange API (all client-side). The source is on GitHub.
